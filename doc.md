# Descrição do Sistema de Coleta de Informações da Máquina

## Visão Geral
Este sistema é responsável por coletar informações relevantes da máquina onde está sendo executado.

## Informações Coletadas
O sistema pode capturar diferentes tipos de dados, incluindo:
- **Informações do Sistema Operacional**
  - Nome e versão do sistema operacional
  - Arquitetura (32-bit ou 64-bit)
  - Tempo de atividade (uptime)

- **Hardware**
  - Modelo do processador (CPU)
  - Quantidade de memória RAM
  - Espaço em disco disponível e total

## Para Rodar
